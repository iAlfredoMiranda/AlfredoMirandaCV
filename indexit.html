Le mie abilità

Front-End

Come sviluppatore Front-End, ho esperienza di 4 anni lavorando con HTML/CSS/JS e frameworks come Bootstrap, Angular, React e preprocessori CSS (LESS/SASS). Oltre a lavorare con il Web Design sviluppo di layouts fatto con Photoshop, Figma per pagine responsive con tecniche SEO applicate.

Web Design

Tutti Front-End deve avere un po' di de Web Designer, ho lavorato 2 anni come insegnante di Web Design insegnando agli studenti a utilizzare Adobe Suite (Photoshop, Illustrator, XD, Premiere e Dreamweaver) per creare layout di pagina, creare banner e arti digitali. Ho l'obiettivo di diventare un giorno uno sviluppatore Full-Stack ed è per questo che sto studiando PHP con Laravel e PostgreSQL per lo sviluppo completo di un sistema web.

Management

Come sviluppatore che sempre ho bisogno gestire il progetto di i mio lavori. Ho sempre cercato usare metodologia agile di sviluppo incentrato sull'adattabilità, produttività e comunicazione utilizatto principalmente il Scrum e il Lean. Tutti per gestire progetti in squadra con 3-5 persone.
